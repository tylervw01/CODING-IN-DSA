Unit 7: Graphs

1. Basic Concepts

A graph is a non-linear data structure consisting of vertices (nodes) and edges (connections). Graphs can be directed or undirected, weighted or unweighted.

2. Representation

  • Adjacency Matrix: 2D array storing edge information.
  • Adjacency List: Each vertex stores a list of connected vertices.

3. Traversal Algorithms

  • Breadth-First Search (BFS): Level-order traversal using a queue. O(V+E).
  • Depth-First Search (DFS): Explore as deep as possible using recursion/stack. O(V+E).

4. Shortest Path Algorithms

  • Dijkstra’s Algorithm: Finds shortest path in weighted graphs with non-negative edges.
  • Bellman-Ford Algorithm: Handles graphs with negative weights.
  • Floyd-Warshall Algorithm: Computes shortest paths between all pairs of vertices.

5. Minimum Spanning Tree (MST)

  • Kruskal’s Algorithm: Greedy approach using edge sorting.
  • Prim’s Algorithm: Greedy approach expanding from a starting vertex.

6. Applications

  • Social networks (friend connections).
  • Routing in communication networks.
  • Pathfinding in maps and games.
  • Dependency resolution in compilers.
