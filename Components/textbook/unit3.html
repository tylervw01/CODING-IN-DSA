Unit 3: Arrays and Pointers

1. Linear Arrays

Arrays are contiguous memory structures that store elements of the same type. They allow constant-time access using indices but have fixed size.

2. Traversing, Insertion, Deletion

  • Traversing: Visiting each element sequentially.
  • Insertion: Adding an element at a position (may require shifting).
  • Deletion: Removing an element (requires shifting to fill gaps).

3. Matrix Multiplication

Arrays can represent matrices. Multiplication involves row-by-column operations. Time complexity is O(n³) for naive methods, but optimized algorithms (Strassen, etc.) reduce complexity.

4. Sorting (Bubble Sort)

Bubble sort repeatedly swaps adjacent elements if they are in the wrong order.
Complexity: O(n²) worst case, O(n) best case (already sorted).

5. Searching (Linear & Binary)

  • Linear search: Scan each element until found. O(n).
  • Binary search: Requires sorted array. Divide and conquer. O(log n).

6. Multidimensional Arrays

Arrays with more than one dimension (e.g., 2D arrays for matrices). Stored in row-major or column-major order in memory.

7. Pointer Arrays

Arrays of pointers allow dynamic referencing of memory locations. Useful for strings, linked structures, and dynamic allocation.
