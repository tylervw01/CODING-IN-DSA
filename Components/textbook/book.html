DSA 2.0 INDEX

Unit 1: Introduction

Overview of data structures, abstract data types, and classification (linear vs non-linear, static vs dynamic).

Unit 2: Strings & Pattern Matching

String storage methods, operations (substring, indexing, concatenation), and algorithms for pattern matching.

Unit 3: Arrays and Pointers

Linear arrays, traversals, insertion/deletion, matrix operations, sorting, searching, and pointer arrays.

Unit 4: Linked Lists

Traversal, insertion, deletion, and header lists (grounded and circular) with dynamic memory applications.

Unit 5: Stacks and Queues

Stack operations (push/pop), queue operations (enqueue/dequeue), and variants like priority queues.

Unit 6: Trees

Tree terminology, binary tree traversals, BST operations, and balanced trees for efficient searching.

Unit 7: Graphs

Directed/undirected graphs, adjacency representations, BFS/DFS traversals, and shortest path algorithms.

Unit 8: Sorting and Searching

Sorting algorithms (merge, radix, quick), searching techniques (binary, hashing), and collision resolution.

Unit 9: AVL Trees

Balanced BSTs with rotations (LL, RR, LR, RL), insertion/deletion, and maintaining height balance.

Unit 10: Warshall’s Algorithm & Shortest Path

Warshall’s path matrix, shortest path algorithms, and applications in weighted graphs and digraphs.
