Unit 5: Stacks and Queues

1. Stack (LIFO)

A stack is a linear data structure that follows the Last-In-First-Out principle. Operations include:

  • Push: Insert an element at the top.
  • Pop: Remove the top element.
  • Peek: View the top element without removing it.

Applications: Function calls, expression evaluation, undo operations.

2. Queue (FIFO)

A queue is a linear data structure that follows the First-In-First-Out principle. Operations include:

  • Enqueue: Insert an element at the rear.
  • Dequeue: Remove an element from the front.
  • Front/Rear: Access the first or last element.

Applications: Scheduling, buffering, resource management.

3. Variants

  • Circular Queue: Connects rear to front for efficient space use.
  • Double-Ended Queue (Deque): Insert/remove from both ends.
  • Priority Queue: Elements dequeued based on priority, not order.

4. Complexity

Stack and queue operations (push, pop, enqueue, dequeue) are O(1). Searching is O(n).
