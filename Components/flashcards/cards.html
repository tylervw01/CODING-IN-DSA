DSA 2.0 Flashcards:

What is an Abstract Data Type (ADT)?

A logical description of operations, independent of implementation.

What is the worst‑case complexity of naive string matching?

O(n·m), where n = text length, m = pattern length.

Which search algorithm requires a sorted array?

Binary Search (O(log n)).

What is the time complexity of Bubble Sort?

O(n²).

What is the advantage of circular linked lists?

Efficient traversal without null termination; useful for queues.

Which data structure is used implicitly in recursion?

Stack (LIFO).

What is the difference between Queue and Stack?

Queue = FIFO, Stack = LIFO.

Which traversal of a BST gives sorted order?

Inorder Traversal.

Which algorithm finds shortest path in unweighted graphs?

BFS (Breadth‑First Search).

What is the complexity of DFS/BFS?

O(V + E), where V = vertices, E = edges.

What is the average case complexity of Quick Sort?

O(n log n).

What is hashing used for?

Constant‑time search and retrieval (average O(1)).

What is the balance factor in AVL Trees?

Height(left subtree) − Height(right subtree).

What is the complexity of Floyd‑Warshall algorithm?

O(n³).

What does Warshall’s algorithm compute?

Transitive closure (path matrix) of a graph.
